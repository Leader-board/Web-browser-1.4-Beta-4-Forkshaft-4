Ribbon Backstage

The Ribbon Backstage is the 'backstage' of the browser , where you can view the AppStore , view the basic website and system information , and the About screen(and the contributors)

To access there , on the Ribbon, just click the logo. This is not available for Menu.

 

Appearance options

The Ribbon is well customisable , including the ability to change the tab type(introduced in 1.4 Beta 3) and tab colour(introduced in 1.4 Beta 4 Forkshaft 2)

The tabs on the Ribbon Backstage

The Ribbon has 4 tabs. They are:-

  1. The Home tab

    This tab is the main tab - in which you can launch the AppStore and view the pinned apps on the store(not fully working)
    Build 93(1.4 Beta 4 Forkshaft 1) didn't have the AppStore, but instead had a UNCHK link and Weather stub(removed)
  2. The Web Info tab
    This tab is the tab in which the main details of the webpage is displayed. In this tab, you can see the domain , website type , encryption level , HTML code and more.



  3. The System Info tab
    This tab shows the details about your PC , the computer name , RAM , network details and a few more.
  4. The About tab
    In this tab , you can see About the browser , view the contributors and see the Trident version.
