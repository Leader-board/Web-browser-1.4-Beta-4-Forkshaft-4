The Menu


This feature was introduced in 1.2.

The Menu is an alternative interface introduced in Leadeboard Web Browser 1.2 . Since the introduction of Ribbon (Leaderboard Web Browser 1.4 Beta 1) , it is used as an alternative interface for those who hate the Ribbon.

To get the Menu from the Ribbon , simply go to Options>Ribbon/Menu options and select the Menu option when prompted.

The Ribbon Backstage in the Ribbon is covered by the Main tab in the Menu instead. 

The Main tab

This tab is like the 'Ribbon Backstage' on the Ribbon .
