The Ribbon


This feature was introduced in 1.4 Beta 1.

The Ribbon is one of the most important area of Leaderboard Web Browser. Introduced in 1.4 Beta 1, it provides an initutive interface on which you can access most of it's features and settings.

Don't like the Ribbon? You can opt for the alternative Menu interface. You can view the Menu help Menu help

 

As you can see from the picure above , the Ribbon is a cusomized variant of Microsoft Office's Ribbon.

Purpose of the tabs

  1. The Home tab. This is the important tab and it is the tab that controls the main navigation of the browser.

  2. The Options tab. In this tab , you can customize many of the options available on Leaderboard Web Browser.

  3. The History/Favourites tab - In this tab , you cna view the History and Favourites that you have selected.

  4. The Tools tab. The purpose of this tab is similar to the Options Tab/

  5. The Help tab. In this tab , you can see the general information about the browser and also re-run the First Run Wizard. You can additionally disable the ToolTips as well.

  6. The Exit tab. In this tab, you can close the browser
