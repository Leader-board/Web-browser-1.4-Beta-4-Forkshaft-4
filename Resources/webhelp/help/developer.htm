The Developer's Guide for Leaderboard Web Browser

 

Extensions can be developed for the browser. The browser accepts 2 types of extensions as of Build 96 :-

  1. Program-based extensions - These types of extensions are those which utilize a separate program to do all it's tasks and it is not much integrated. Message Maker falls under this group.
  2. Embedded extensions - These types of extensions are extensions that modify/embed into the web browser's main form(need not be always the main form). These types of extensions may use a separate program(like for settings).

You should develop extensions using Visual Basic only for this browser. An extension must meet the following requirements

  1. The extension should be compartable with NET 3.0(through they can be lower than that)
  2. The extension must match/be below the system requirements for the browser.
  3. The extension should work with the browser
  4. The extension must not damage/impair the core functions of the browser(through they can be modified to some extent)

Through we recommend you to use Visual Studio 2005/2008 for developing extensions, VB6 is also partially allowed.
